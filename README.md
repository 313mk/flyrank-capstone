# Resilient Frontend AI Portfolio

A modern, production-grade portfolio demonstrating resilient client-side state management, real-time LLM stream handling, and secure API integration.

## Tech Stack
- **Framework:** React + TypeScript (Vite)
- **Styling:** Tailwind CSS
- **AI Tooling:** Claude.ai / Cursor IDE

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) 20 LTS (or 18 LTS)
- npm 9+

### Local development

```bash
git remote add origin https://github.com/313mk/flyrank-capstone.git
cd flyrank-capstone
npm install
npm run dev
